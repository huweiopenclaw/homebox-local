## SearchAPI MCP 工具配置

已安装并配置 SearchAPI MCP 工具用于网络搜索，替代 Brave Search。

### 配置位置
- **MCP 服务器**: `C:\Users\1990h\.openclaw\tools\searchAPI-mcp`
- **mcporter 配置**: `C:\Users\1990h\.openclaw\config\mcporter.json`
- **环境变量**: `C:\Users\1990h\.openclaw\tools\searchAPI-mcp\.env`

### 使用方式
通过 mcporter CLI 调用：
```bash
npx mcporter call searchapi.search_google q:"搜索内容" --config C:\Users\1990h\.openclaw\config\mcporter.json
```

### 可用工具
- `search_google` - Google 网页搜索
- `search_google_videos` - 视频搜索
- `search_google_ai_mode` - AI 模式搜索
- `search_google_maps` - 地图搜索
- `search_google_events` - 活动搜索
- `search_google_flights` - 航班搜索
- `search_google_hotels` - 酒店搜索
- `health_check` - 健康检查
- 等 15 个工具
