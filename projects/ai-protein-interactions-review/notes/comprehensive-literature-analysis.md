### 2.2 Leclercq (2026) - pLM 应用框架

**核心贡献**: 蛋白质语言模型在 PPI 中的应用全景

```
pLM 预训练 → 嵌入表示 → 下游任务
     ↓            ↓          ↓
UniRef/UniProt  高维向量   PPI/亲和力预测
```

**关键模型**:
- ESM-2: 广泛下游应用
- ProtGPT2: 序列生成
- MINT: 多聚体相互作用 Transformer
- ProBASS: 结合亲和力预测

**PPI 相关发现**:
- MINT: AUPRC ~0.69，29% 准确性提升
- ProBASS: r ≈ 0.8 结合亲和力预测
- pLM 嵌入可有效替代手工特征

### 2.3 Deng (2026) - GNN 三维框架

**核心贡献**: 图构建 → 架构 → 应用的统一框架

```
图构建 → GNN 架构 → 生物应用
   ↓         ↓          ↓
PPI/GRN   GCN/GAT/   疾病基因/
分子图    GraphSAGE   药物发现
```

**图构建策略**:
1. Graph Structure Learning (GSL)
2. Curvature Rewiring
3. Effective Resistance Optimization
4. Edge Removal Sampling

**PPI 网络建模**:
- 节点 = 蛋白质
- 边 = 物理/功能相互作用
- 任务: 疾病基因预测、靶点识别

### 2.4 Li (2026) - AlphaFold 整合框架

**核心贡献**: AlphaFold 与实验技术的协同分析

```
AlphaFold 预测 → 初始模型 → cryo-EM/ET 验证 → 最终结构
```

**统计数据**:
- 78.1% 研究使用 AlphaFold
- 使用分辨率范围: 3.0-31.7 Å

**三种使用模式**:
1. 初始模型生成
2. 结构域拟合/验证
3. 柔性精修/界面分析

**局限性**:
- 静态构象预测
- IDR 预测困难
- 瞬时相互作用不准
- 跨膜蛋白偏差

### 2.5 Strom (2025) - AF-MM 批判性视角

**核心贡献**: 揭示 AlphaFold-Multimer 的数据偏差

```
训练数据偏差 → 有序区域偏向 → 无序界面预测不准
```

**关键问题**:
- 训练数据偏向有序区域相互作用
- 基准测试同样存在偏差
- 无序区域界面预测能力不足

### 2.6 Subedy (2026) - 表示学习视角

**核心贡献**: ML 表示与物理先验的整合

序列/结构 → ML 表示 → 物理先验整合 → 可解释预测

**关键观点**:

## 三、方法演进脉络

### 3.1 时间线

```
2020          2022          2024          2026
  |             |             |             |
AlphaFold → AlphaFold2 → AF-Multimer → AlphaFold3
  |             |             |             |
传统ML    →    pLM兴起   →   GNN整合   →   多模态融合
  |             |             |             |
手工特征  →   自监督学习  →  端到端预测  →  物理约束整合
```

### 3.2 范式转变

| 阶段 | 特征 | 代表方法 |
|------|------|----------|
| **传统ML** | 手工特征 + 分类器 | SVM, RF |
| **早期DL** | 自动特征提取 | CNN, RNN |
| **pLM时代** | 序列嵌入 | ESM, ProtTrans |
| **结构预测革命** | 端到端结构 | AlphaFold 系列 |
| **图学习时代** | 关系建模 | GNN, GAT |
| **多模态融合** | 整合多种信息 | AF3, MINT |
