### 核心框架: 四层结构

```
生物图构建 → GNN 架构 → 系统化基准 → 领域应用
```

### 生物网络类型

| 网络类型 | 节点 | 边 | 典型任务 |
|----------|------|-----|----------|
| **PPI** | 蛋白质 | 相互作用 | 疾病基因、靶点识别 |
| **GRN** | 基因/调控因子 | 激活/抑制 | 细胞命运、疾病失调 |
| **分子图** | 原子/残基 | 共价键 | 性质预测、药物筛选 |
| **知识图谱** | 异构实体 | 跨域关系 | 药物重定位、机制阐明 |

### 图结构优化方法

| 方法 | 原理 | 公式 |
|------|------|------|
| **GSL** | 邻接矩阵可学习 | $A^* = g(A, \tilde{A}_\theta(X, Z))$ |
| **Curvature Rewiring** | Jost-Liu 曲率 | $JLC(i,j) = \frac{2}{d_i} + \frac{2}{d_j} - 2 + ...$ |
| **Effective Resistance** | 最小化总电阻 | $\min_E \sum R_G(i,j)$ |

### GNN 架构对比

| 特性 | 谱域方法 | 空间域方法 |
|------|----------|------------|

## 三、文献 3: Li et al. (2026) - AlphaFold 优势与局限

**文件**: `s10930-025-10310-8.pdf`
**期刊**: The Protein Journal

### 研究方法

- **数据来源**: EMDB (电子显微镜数据库)
- **时间范围**: 2024年沉积条目 (2022-2025 发表)
- **样本量**: 32 项研究
- **筛选标准**: Cryo-EM/ET 基础、同行评审、方法细节完整

### 关键统计数据

| 指标 | 数值 |
|------|------|
| 总研究数 | 32 |
| AlphaFold 使用率 | **78.1%** (25/32) |
| 未使用率 | 21.9% (7/32) |
| 分辨率范围 | 3.0 Å - 31.7 Å |
| 原子精度残基比例 | ~67% (1/3 不达标) |

### 分辨率与 AF 使用关联性

```
Chi-square: χ² = 0.20, p = 0.656
Fisher's exact: p = 0.640
Spearman's ρ: ρ = -0.163, p = 0.373
```
**结论**: 无显著关联，AF 使用由局部可解释性驱动

### PPI 应用案例

| 复合物 | EMD ID | 分辨率 | AF 贡献 |
|--------|--------|--------|---------|
| E3 泛素连接酶-底物 | EMD-38163 | 12.0 Å | 揭示疏水-静电界面 |
| 染色质重塑复合物 | EMD-37966 | 9.7 Å | 远端结构域变构耦合 |
| 痘病毒核心蛋白 | EMD-17704/17708/17753 | 7.7-13.4 Å | A10-A4 异二聚体支架 |
| 核孔复合物 | EMD-19137 | 30 Å | ORF6 阻断核输入机制 |
| 狂犬病毒基质蛋白 | EMD-46612 | 18.7 Å | M:N 1:1 化学计量 |

### AlphaFold 局限性汇总

| 局限类型 | 具体表现 |
|----------|----------|
| **静态 vs 动态** | 无法捕捉构象集合/状态转换 |
| **IDRs** | pLDDT < 50，碎片化预测 |
| **配体/辅因子** | 无法预测配体诱导构象变化 |
| **膜蛋白** | 螺旋位置误差 3-5 Å |
| **原位组装** | 无法推断介观尺度组织 |
| **瞬态相互作用** | 弱/柔性界面预测困难 |
| **非蛋白组分** | 核酸、脂质无法直接预测 |

### 整合工作流

```
AF 预测 → 初始模型 → 密度拟合 → 柔性精修 → 正交验证
    ↓         ↓          ↓          ↓          ↓
pLDDT筛选  ChimeraX   Phenix     ISOLDE    XL-MS/HDX-MS
```
