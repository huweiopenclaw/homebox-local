## 二、PPI 预测方法深度对比

### 2.1 方法分类体系 (Alkhateeb 框架)

**双任务分类**:
- **Approach A (位点预测)**: 预测孤立蛋白表面的相互作用残基

| 训练偏差 | 有序区域偏向 | 无序界面预测不足 |

## 六、蛋白质语言模型与 PPI

### 6.1 主要 pLM 模型 (Leclercq)

| 模型 | 参数量 | 训练数据 | PPI 相关应用 |
|------|--------|----------|--------------|
| ESM-2 | 650M | 6.14亿序列 | 嵌入用于下游任务 |
| ProtBERT | - | UniProt | 二级结构、PPI |
| ProtT5 | - | 大规模 | 序列-结构联合 |
| ProGen2 | **6.4B** | >10亿序列 | 序列生成 |
| MINT | ESM-2扩展 | STRING | 多蛋白联合处理 |
| IgBERT/IgT5 | - | >20亿抗体链 | 抗体-抗原结合 |

### 6.2 pLM 在 PPI 中的关键优势

1. **端到端学习**: 从序列直接预测相互作用
2. **长程依赖**: Transformer 捕获远距离残基关系
3. **零样本泛化**: 无需任务特定训练
4. **嵌入表示**: 高维向量编码功能信号

### 6.3 pLM 挑战

| 挑战 | 描述 |
|------|------|
| 可解释性 | 黑盒模型，难以生物学解释 |
| 数据偏差 | 训练数据偏向特定物种/家族 |
| 训练-测试泄漏 | 序列重叠导致泛化高估 |
| 计算成本 | 大模型需要大量资源 |

## 十、本综述差异化定位建议

### 10.1 核心差异化点

1. **"Mapping + Modeling" 双重视角**
   - 不仅预测，更关注系统性映射 PPI 空间
   - 结合网络分析和结构预测

2. **方法演进与范式转变**
   - 从传统 ML → DL → pLM → 多模态融合
   - 结构预测革命对 PPI 领域的影响

3. **批判性综合**
   - 不仅总结，更批判性分析方法优劣
   - 明确局限性和适用边界

4. **应用导向结构**

   - 每种应用的方法选择指导

5. **整合性框架**
