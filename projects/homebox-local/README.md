# HomeBox Local

> 🏠 纯本地版智能家庭收纳助手 | 无服务器架构，数据完全在用户手机上

## ✨ 特点

- 🔒 **极致隐私** - 数据 100% 本地存储，从不离开你的设备
- 💰 **零运维成本** - 无需服务器、无需云存储费用
- ⚡ **极速响应** - 本地数据库查询，毫秒级响应
- 📱 **开箱即用** - 下载即用，无需注册账号
- 🌙 **离线友好** - 基础功能完全离线可用
- 🤖 **AI 加持** - 支持 GLM-4V、Claude、GPT-4V 等

## 🎯 功能

| 功能 | 描述 |
|------|------|
| 📸 拍照识别 | 拍照自动识别箱内物品 |
| 📍 位置记录 | AI 识别箱子存放位置 |
| 💬 对话查询 | 自然语言查找物品 |
| 🔍 智能搜索 | 关键词 + 位置筛选 |
| 💾 数据备份 | 导出/导入数据文件 |
| ⚙️ 自定义 AI | 配置自己的 API Key |

## 🛠️ 技术栈

- **框架**: Flutter 3.27+
- **数据库**: SQLite
- **状态管理**: Riverpod
- **AI 服务**: GLM-4V / Claude / GPT-4V

## 📱 支持平台

- ✅ Android
- ✅ iOS
- ✅ 鸿蒙 (HarmonyOS)

## 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/huweiopenclaw/homebox-local.git

# 进入项目目录
cd homebox-local

# 安装依赖
flutter pub get

# 运行
flutter run
```
