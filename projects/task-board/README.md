# 📌 任务看板

HOC & 主人的任务追踪系统

## 功能特性

- ✅ 看板视图：待办 / 进行中 / 已完成 / 已取消
- ✅ 任务管理：创建、编辑、删除任务
- ✅ 状态切换：拖拽式状态更新
- ✅ 负责人分配：HOC 或 主人
- ✅ 优先级标记：低 / 中 / 高 / 紧急
- ✅ 本地存储：数据保存在浏览器，刷新不丢失

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

## 技术栈

- **框架**: Next.js 16 (App Router)
- **样式**: Tailwind CSS
- **存储**: 浏览器 LocalStorage
- **语言**: TypeScript

## 任务状态说明

| 状态 | 说明 |
|------|------|
| 📋 待办 | 等待开始的任务 |
| 🔄 进行中 | 正在执行的任务 |
| ✅ 已完成 | 已完成的任务 |
| ❌ 已取消 | 已取消的任务 |

## 优先级说明

| 优先级 | 颜色 | 说明 |
|--------|------|------|
| 低 | 灰色 | 可延后处理 |
| 中 | 黄色 | 正常优先级 |
| 高 | 橙色 | 需要优先处理 |
| 紧急 | 红色 | 立即处理 |

## Convex 云数据库 (可选)

如需使用 Convex 云数据库（支持多设备同步）：

1. 安装 Convex CLI: `npm install convex`
2. 登录 Convex: `npx convex login`
3. 启动开发: `npx convex dev`
4. 配置环境变量: `NEXT_PUBLIC_CONVEX_URL`
